# BS_PIDNET
BSnet version of PIDNet for semantic image segmentation

Look at the original PIDNet github

[Github](https://github.com/XuJiacong/PIDNet)

My one is the modification of the original.
My network is convex and has error correction abilities during training.

Please read my paper for more details.

[My paper]
(https://zenodo.org/records/19496155)

[Other paper]
(https://zenodo.org/records/19402689)
